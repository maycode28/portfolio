# May Kim Portfolio

GitHub Pages에 바로 올릴 수 있는 정적 포트폴리오 사이트입니다.

## 로컬 확인

브라우저에서 `index.html`을 직접 열면 됩니다.

## GitHub Pages 배포

1. GitHub에서 새 저장소를 만듭니다. 예: `maykim.github.io` 또는 `portfolio`
2. 이 폴더에서 Git을 초기화하고 커밋합니다.
3. 원격 저장소를 연결한 뒤 `main` 브랜치로 push합니다.
4. GitHub 저장소의 `Settings > Pages`에서 Source를 `Deploy from a branch`, Branch를 `main`, Folder를 `/root`로 선택합니다.
5. 저장하면 몇 분 뒤 Pages 주소가 생성됩니다.

## 수정할 곳

- `index.html`: 소개, 기술, 프로젝트, 연락처 내용
- `styles.css`: 전체 디자인
- `smiletrack.html`, `orda.html`, `oopslog.html`: 프로젝트 상세 페이지
