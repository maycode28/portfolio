AI Service

oopsLog

실패를 기록하는 순간, 생각이 바뀌는 경험

사용자가 실패나 부정적인 경험을 기록하면 AI가 사실과 감정을 분리하고, 인지 왜곡을 탐지하며, 더 현실적인 사고 재구성을 돕는 서비스입니다.

AI Stack

Primary AI Engine: Gemini

Fallback System: OpenAI API

Core Features

Fact Extraction

주관적 감정과 객관적 사실을 분리합니다.

Distortion Detection

흑백논리, 과잉일반화 등 인지 왜곡 패턴을 식별합니다.

Reframing

더 건설적이고 현실적인 관점의 대안을 제시합니다.

JSON 구조 응답

AI 분석 결과를 정형화해 시스템 연동성을 높입니다.

Output Example

{
  "fact_extraction": {
    "subject": "Presentation failure",
    "facts": ["Missed deadline", "Client requested changes"]
  },
  "distortion": "All-or-nothing thinking",
  "reframing": "A delay does not mean the project is a failure."
}
